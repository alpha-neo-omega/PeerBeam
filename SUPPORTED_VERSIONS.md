# Supported Versions

PeerBeam is pre-1.0 (currently `0.2.x`, Beta). Only the latest minor series
receives fixes.

| Version | Supported |
|---|---|
| 0.2.x | ✅ (current, Beta) |
| < 0.2 | ❌ |

Pre-1.0: minor versions may include breaking changes (documented in
[CHANGELOG.md](CHANGELOG.md) and [docs/MIGRATION.md](docs/MIGRATION.md)). The FFI
C-ABI is versioned separately (`pb_abi_version`, currently `1`); breaking ABI
changes bump it.
