# Bundled fonts

**Google Sans Flex** — static weight instances (400/500/600/700) obtained from
the official Google Fonts service (family: "Google Sans Flex").

Google Fonts families are distributed under the SIL Open Font License 1.1
(https://openfontlicense.org). The font remains under its own license; it is
bundled unmodified for offline use — PeerBeam performs no runtime font
fetching.
